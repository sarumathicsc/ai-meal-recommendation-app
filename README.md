# AI Meal Recommendation App

## Overview

The AI Meal Recommendation App is an Android application developed to help users discover meal suggestions based on their preferences. The project focuses on building an intuitive mobile experience while exploring recommendation-based application design.

This project strengthened my understanding of Android application development, user interface design, and mobile application architecture.

---

## Features

- Personalized meal recommendations
- Simple and user-friendly interface
- Interactive mobile experience
- Easy navigation
- Android-based application

## Installation

### Clone the repository

```bash
git clone https://github.com/sarumathicsc/ai-meal-recommendation-app.git
```

### Open the project

Open the project using **Android Studio**.

### Build the project

Allow Gradle to sync automatically.

### Run the application

Connect an Android device or start an Android Emulator, then click **Run** in Android Studio.
